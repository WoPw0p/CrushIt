# CrushIt — Creative Brief

## Hook
**CrushIt** 是一台信号破坏引擎。它不是让声音变"脏"，而是让声音被数字系统本身吞噬——时空断裂、位深崩塌、浮点错误被放大成武器。

## Description
CrushIt 包含六个独立的破坏性处理模块，可以按任意顺序组合：

### 模块 1: 时空畸变 (Temporal Warp)
持续录入音频到环形缓冲区，读取头不按线性移动，而是根据瞬态检测或音高分析进行跳跃。当检测到鼓点或其他强瞬态时，读取头瞬间回跳指定毫秒数，并在回跳点执行极短循环（Micro-looping），产生类似机器卡顿、CD 跳碟的嗡鸣声（Stutter/Buzz）。

### 模块 2: 比特压缩 (Bit Crusher)
经典的位深压缩——将音频量化到指定低位深，引入阶梯失真和量化噪声。

### 模块 3: 采样率腰斩 (Sample Rate Decimator)
降低采样率，引入混叠伪影。通过零阶保持或线性插值实现降采样，在高频处产生镜像频率。

### 模块 4: 波形裁剪 (Waveform Clipper)
提供 Hard Clip、Soft Clip、Wavefold 三种模式的前置增益+削波，从温和饱和到硬性切除。

### 模块 5: 数字噪点 (Digital Debris)
同步切片触发的白噪音/数字爆音模拟。可锁定宿主节拍或自由运行，模拟信号断裂感——数据包丢失、D/A 故障、时钟抖动。

### 模块 6: 位反转 (Bit Flipping — 浮点错误注入)
本模块是 CrushIt 的标志性功能。它不按传统的整数位压缩方式工作，而是直接操作 IEEE 754 单精度浮点数的位模式：
- 选择目标 Bit 位（符号位、指数位、或尾数位）进行强制置反
- 随机偏移模式：对采样值乘以巨大非法数字（如 ×1000），然后通过硬限幅器压回可听范围
- 结果从微妙的数字颗粒感延伸到完全的信号崩溃

## Vibe
有质感的科技风格（Textured Sci-Fi / Industrial Tech）。深色背景 + 冷色调 LED，带扫描线或微小噪点纹理，界面像是一台被修复的故障设备诊断终端。
